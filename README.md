# Deezer Artist Search

- Data from [Deezer API](https://developers.deezer.com/api)
- Made using Material UI components
- App created with Vite

## Getting Started

### Installing

```
npm install
```

### Executing program

```
npm run dev
```

### Building program

```
npm run build
```

### Preview build

```
npm run preview
```
